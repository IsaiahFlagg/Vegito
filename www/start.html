How to work on github projects

Clone Repo
  • Go to www.github.com
  • Click green button
  • clone with SSH
Step 2: Make some changes to the code then,
  • git status
  • git add .
  • git status
  • git commit -m "This is needed" -m "This second part not so much. Use this when writing a really long message that someone might read and someone might not read blah blah blah"
  • git push origin master

Step 2 is repeated until the project is done. It's like a save button on the playstation
